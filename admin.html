Welcome, admin!

This is the database center for Ali Baba's Fabulous Donuts.
All donuts currently for sale are listed below, ordered by their ID in the database.

You can change the selection by using the prompts and buttons below. Note that every donut has a unique ID number, a name, a description, and a price (in US dollars, decimal format; please write as 2.99, not $2.99).



For the following buttons:

  • Add: input a name, description, and price. The new donut's ID is automatically added.
  • Update: input an ID, then give the updated name, description, and price for the corresponding donut. Those values are used to update the donut's info. (Inputs left blank will remain unchanged.)
  • Delete: input just the ID. The corresponding donut will be deleted from the list.
